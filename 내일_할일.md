### 테스트 체크리스트

**1. 캐릭터 생성**
- [ ] 새 캐릭터 만들기
- [ ] 이름 입력 (텍스트 잘림 없는지 확인)

**2. 종족 선택**
- [ ] 필보르 선택
- [ ] LUK 능력치에 "+3" 표시되는지 확인
- [ ] 합계 "=9", 보너스 "[3]" 표시 확인

**3. 클래스 선택**
- [ ] 메인 클래스: 워리어
- [ ] 서포트 클래스: 메이지
- [ ] HP, MP 자동 계산되는지 확인
- [ ] 능력치에 클래스 보너스 적용되는지 확인

**4. 라이프 패스**
- [ ] 출신 주사위 굴리기 → "24" 같은 연결 표기 확인
- [ ] 드롭다운에 자동 선택되는지 확인
- [ ] 경위, 목적도 같은 방식으로 테스트

**5. 레벨/성장점**
- [ ] 레벨 +/- 버튼으로 레벨 올리기/내리기
- [ ] 성장점 자동 계산 (레벨-1 × 2) 확인
- [ ] 능력치 +/- 버튼으로 성장점 투자
- [ ] 남은 성장점 감소하는지 확인

**6. 스킬 시스템** ⭐ NEW!
- [ ] 스킬 탭 클릭 (탭 작동 확인)
- [ ] **+ 버튼 클릭**
- [ ] 스킬 선택 다이얼로그 열리는지 확인
- [ ] 필터 드롭다운 정상 작동하는지 (메인 클래스만/서포트 클래스만/일반/전체)
- [ ] 검색 기능 테스트
- [ ] 스킬 선택하여 획득
- [ ] 스킬 레벨 +/- 버튼 테스트
