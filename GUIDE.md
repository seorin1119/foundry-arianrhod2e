## 1. 시작하기

### 월드 생성
1. Foundry VTT **Setup** 화면에서 **Game Worlds → Create World** 클릭
2. **Game System**에서 `Arianrhod RPG 2E` 선택
3. 월드 이름을 입력하고 생성

### 첫 로드
- GM이 처음 로드하면 **컴펜디움 팩**이 자동 생성됩니다 (스킬 330개, 장비, 에너미 스킬)
- 이 과정은 한 번만 실행되며, 이후에는 자동으로 건너뜁니다

## 2. 캐릭터 만들기

### 액터 생성
1. 사이드바 **Actors** 탭 → **Create Actor** 클릭
2. 타입을 `Character`로 선택

### 기본 정보 입력
캐릭터 시트 상단 헤더에서:
- **종족**: 휴린, 엘다나안, 네베르프, 필볼, 바르나 중 선택
- **메인 클래스**: 워리어, 어콜라이트, 메이지, 시프
- **서포트 클래스**: 메인 클래스 포함 건슬링어, 사무라이, 세이지, 닌자, 알케미스트 등
- **레벨**: `-` / `+` 버튼으로 조정
- **길드**: 소속 길드명 입력

### 능력치 탭
7개 능력치를 입력합니다:

| 능력치 | 약칭 | 설명 |
|--------|------|------|
| 근력 (STR) | STR | 물리 공격력, 운반 능력 |
| 기민 (DEX) | DEX | 명중, 수작업 |
| 민첩 (AGI) | AGI | 회피, 행동 순서 |
| 지력 (INT) | INT | 마법 공격력, 지식 |
| 감각 (SEN) | SEN | 지각, 탐지 |
| 정신 (MEN) | MEN | 마법 방어, 의지 |
| 행운 (LUK) | LUK | 행운 판정, 드롭 |

- 각 능력치의 **기본값**과 **보정치**를 입력하면 **전투 능력치**가 자동 계산됩니다
- **HP**, **MP**: 클래스/레벨에 맞게 입력
- **페이트**: 최대 5개, 별 아이콘 클릭으로 현재값 조정

### 탭 구성
| 탭 | 내용 |
|----|------|
| **능력치** | 7대 능력치, HP/MP, 페이트, 전투 스탯 |
| **장비/소지품** | 무기, 방어구, 장신구, 소지품, 골드 |
| **스킬** | 취득한 스킬 목록, 발동 |
| **연결/성장** | 커넥션, 성장점 |
| **프로필** | 외모, 배경, 메모 |

## 3. 스킬

### 스킬 추가
1. **스킬 탭** → `+` 버튼 클릭
2. **스킬 선택 다이얼로그**가 열립니다
3. 클래스별 필터로 사용 가능한 스킬을 확인
4. 원하는 스킬을 클릭하면 캐릭터에 추가됩니다

### 스킬 발동
- 스킬 옆의 **발동 버튼** (주사위 아이콘)을 클릭
- MP가 충분하면 자동으로 소비되고 **채팅 카드**에 결과가 표시됩니다
- MP가 부족하면 경고 메시지가 나타납니다

### 스킬 라이브러리
- 시스템에 **330개 스킬**이 내장되어 있습니다
- 메인 4클래스 + 서포트 클래스별로 분류
- 각 스킬에는 타이밍, 판정, 대상, 사거리, 코스트, 효과 정보가 포함

---

## 4. 장비와 소지품

### 장비 추가
- **장비/소지품 탭**에서 각 슬롯의 `+` 버튼 클릭
- 장비 선택 다이얼로그에서 원하는 아이템 선택
- 또는 컴펜디움에서 드래그앤드롭

### 장비 슬롯
| 슬롯 | 설명 |
|------|------|
| 오른손 / 왼손 | 무기 또는 방패 장착 |
| 머리 | 투구 등 |
| 몸통 / 방어구1 / 방어구2 | 갑옷, 로브 등 |
| 장신구 | 반지, 목걸이 등 |

### 자동 계산
장비를 장착하면 전투 스탯이 자동으로 반영됩니다:
- **명중**: 무기 명중 보정 + 능력치
- **공격력**: 무기 공격력 + 능력치
- **물리방어 / 마법방어**: 방어구 수치 + 능력치
- **행동력**: 장비 수정치 반영

## 5. 전투

### 전투 시작
1. 토큰을 선택하고 **전투 트래커** (사이드바 검 아이콘)에서 **Begin Combat**
2. **이니셔티브**: 기본적으로 캐릭터의 **행동값**이 자동 사용됩니다
   - 시스템 설정에서 "롤 방식" (2D6 + 행동값)으로 변경 가능

### 공격
1. 캐릭터 시트 능력치 탭에서 **공격 판정** 버튼 클릭 (주사위 아이콘)
2. 2D6 + 명중 수정치로 판정
3. 채팅 카드에 결과 표시:
   - **달성치** (2D6 + 수정치)
   - **크리티컬** (6이 2개 이상) / **펌블** (1이 2개 이상) 배지
   - **데미지 롤** 버튼 (명중 시 클릭)

### 데미지
- 공격 카드의 **데미지 롤** 버튼을 클릭하면 데미지가 계산됩니다
- 데미지 카드에는 **데미지 적용** 버튼이 있어 대상에게 바로 적용 가능

### 회피
- 대상 토큰의 시트에서 **회피 판정** 버튼 클릭
- 2D6 + 회피 수정치로 판정

### 토큰 HUD
토큰을 우클릭하면 커스텀 패널이 표시됩니다:
- **HP / MP 바**: 현재 비율 표시
- **±1, ±5 버튼**: HP/MP 빠른 조정
- **명중/회피 판정 버튼**: 빠른 전투 판정

## 6. 상태이상

시스템에 13종 상태이상이 내장되어 있습니다:

| 상태이상 | 효과 |
|----------|------|
| 독 (Poison) | 라운드 종료 시 데미지 |
| 질병 (Disease) | 능력치 감소 |
| 노화 (Aging) | 능력치 감소 |
| 매혹 (Charm) | 적을 공격 불가 |
| 공포 (Fear) | 행동 제한 |
| 석화 (Petrify) | 행동 불가 |
| 기절 (Stun) | 1라운드 행동 불가 |
| 수면 (Sleep) | 행동 불가, 공격 시 해제 |
| 전투불능 (Incapacitated) | HP 0 |
| 눈멀음 (Blind) | 명중 -4 |
| 침묵 (Silence) | 마법 사용 불가 |
| 출혈 (Bleed) | 라운드 종료 시 데미지 |
| 속박 (Bind) | 이동 불가 |

- 토큰 HUD에서 상태이상 아이콘을 클릭하여 토글
- 활성화된 상태이상은 토큰 위에 아이콘으로 표시됩니다

## 7. 에너미 설정

### 에너미 생성
1. **Actors → Create Actor** → 타입 `Enemy` 선택
2. 이름, 레벨, 능력치 입력
3. 에너미 스킬 선택 UI에서 스킬 추가

### 에너미 스킬
- `+` 버튼을 클릭하면 **에너미 전용 스킬 선택 다이얼로그**가 열립니다
- 에너미 스킬 라이브러리에서 원하는 스킬을 선택하여 추가
- 드래그앤드롭도 지원됩니다

## 8. 길드

### 길드 생성
1. **Actors → Create Actor** → 타입 `Guild` 선택
2. 길드 시트가 열립니다

### 길드 시트 기능
- **길드 레벨**: 레벨 조정 (서포트 슬롯 수에 영향)
- **골드**: 길드 자금 관리
- **길드 마스터**: 마스터 이름 입력
- **멤버 관리**: 길드원 추가/제거
- **서포트**: 길드 서포트 추가 (슬롯 = 길드 레벨 + 2)
  - 55개 서포트 중 선택
  - 중복 가능 여부 자동 체크

## 9. 매크로와 핫바

### 매크로 생성
핫바에 아이템을 드래그앤드롭하면 자동으로 매크로가 생성됩니다:

| 아이템 타입 | 매크로 효과 |
|-------------|-------------|
| **스킬** | 스킬 발동 (MP 소비 포함) |
| **무기** | 공격 판정 (장착 확인 포함) |
| **기타 아이템** | 아이템 시트 열기 |

### 능력치 판정 매크로
스크립트 매크로로 능력치 판정을 실행할 수 있습니다:
```javascript
game.arianrhod2e.macros.rollAbilityCheckMacro("str");  // STR 판정
game.arianrhod2e.macros.rollAbilityCheckMacro("dex");  // DEX 판정
```

사용 가능한 능력치 키: `str`, `dex`, `agi`, `int`, `sen`, `men`, `luk`

## 10. 시스템 설정 (하우스룰)

**Configure Settings → Arianrhod RPG 2E** 카테고리에서 접근합니다.

| 설정 | 기본값 | 설명 |
|------|--------|------|
| **크리티컬 범위** | 6이 2개 이상 | "1개 이상"으로 변경하면 크리티컬 확률 증가 |
| **페이트 사용** | 활성화 | 비활성화 시 판정 다이얼로그에서 페이트 옵션 숨김 |
| **자동 데미지 계산** | 활성화 | 대상의 방어력을 자동으로 차감 |
| **HP 0 자동 전투불능** | 활성화 | HP가 0이 되면 알림 표시 |
| **이니셔티브 방식** | 고정 (행동값) | "롤"로 변경 시 2D6 + 행동값으로 이니셔티브 결정 |

## 11. 캐릭터 내보내기/가져오기

### 내보내기
1. 캐릭터/에너미 시트 헤더 우측의 **내보내기 아이콘** (📤) 클릭
2. JSON 파일이 자동 다운로드됩니다
3. 파일에는 모든 능력치, 스킬, 장비 데이터가 포함됩니다

### 가져오기
1. 캐릭터/에너미 시트 헤더 우측의 **가져오기 아이콘** (📥) 클릭
2. 파일 선택 다이얼로그에서 JSON 파일 선택
3. 시스템 ID와 버전 호환성이 자동 검증됩니다
4. 새 액터가 생성되고 시트가 자동으로 열립니다

## 12. 컴펜디움

시스템에 3개의 컴펜디움 팩이 포함되어 있습니다:

| 팩 | 내용 |
|----|------|
| **Skills** | 330개 스킬 (메인 4클래스 + 서포트 클래스) |
| **Equipment** | 무기, 방어구, 장신구 |
| **Enemy Skills** | 에너미 전용 스킬 |

- 컴펜디움에서 아이템을 캐릭터 시트로 **드래그앤드롭**하여 추가 가능
- GM은 `game.arianrhod2e.resetPack("pack-name")`으로 컴펜디움을 초기화할 수 있습니다

## 13. 팁과 단축키

### 드래그앤드롭
- 컴펜디움 → 캐릭터 시트: 아이템 추가
- 캐릭터 시트 → 핫바: 매크로 생성
- 스킬/장비 간 시트 이동 가능

### 주사위 판정
모든 판정은 **2D6 + 수정치** 기반입니다:
- **크리티컬**: 6이 2개 이상 나오면 자동 성공
- **펌블**: 1이 2개 이상 나오면 자동 실패
- **페이트**: 판정 시 페이트 다이스를 추가로 사용 가능 (설정에서 활성화 시)

### 채팅 카드
- 스킬 발동, 공격, 데미지 판정 결과가 **시각적 채팅 카드**로 표시됩니다
- 카드에 포함된 버튼으로 데미지 롤, 데미지 적용 등을 바로 수행할 수 있습니다

### 유용한 콘솔 명령어 (GM)
```javascript
// 컴펜디움 재생성
game.arianrhod2e.populateAllPacks();

// 능력치 판정
game.arianrhod2e.macros.rollAbilityCheckMacro("str");

// 스킬 매크로
game.arianrhod2e.macros.rollSkillMacro("스킬이름");

// 공격 매크로
game.arianrhod2e.macros.rollAttackMacro("무기이름");
```
